== Description ==

Apprenticeship Connect is a powerful WordPress plugin designed to seamlessly import and manage apprenticeship vacancies from the UK Government API (Display Advert API v2).

Key Features:
*   **Two-Stage Import Engine:** Efficiently fetches vacancy listings and then performs deep-fetch lookups for full details.
*   **Rate-Limit Resilience:** Built-in exponential backoff and 429-awareness to handle API limits gracefully.
*   **Modern Admin Interface:** Professional dashboard with real-time import logging and task management.
*   **Gutenberg & Elementor Ready:** Includes blocks and dynamic tags for displaying vacancies.
*   **Extensible Architecture:** Well-structured codebase using modern PHP patterns.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your API Subscription Key in the plugin settings.
4. Create an Import Task to start fetching vacancies.
